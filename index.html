Étude lexicale et contextuelle du terme 鬼 (oni)

Analyse automatique d’un corpus web japonais

Présentation du projet

Ce projet a pour objectif d’analyser les usages du terme japonais 鬼 (oni), dans ses différentes formes graphiques (kanji, hiragana et romaji), à partir d’un corpus de pages web japonaises.

L’analyse repose sur une collecte automatique des pages, suivie de plusieurs traitements linguistiques :

  • vérification du code HTTP et de l’encodage
  • extraction du HTML brut
  • dump textuel des pages
  • comptage des occurrences du mot cible
  • construction d’un concordancier contextuel

Contenu du tableau d’analyse

Le tableau de résultats présente, pour chaque URL du corpus :

  • l’URL analysée
  • le code HTTP de la page
  • l’encodage détecté ou supposé
  • le nombre total d’occurrences du terme oni
  • un accès au HTML brut
  • un dump textuel de la page
  • un concordancier mettant en évidence les contextes d’apparition
